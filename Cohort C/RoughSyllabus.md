New Age Software Engineering Program - Cohort C

Module 1: Generative-AI Foundations & Prompt Craft
Foundation in LLM architecture, tokenization, and cost economics. Advanced prompting techniques
including Chain-of-Thought, Tree-of-Thought, persona injection, and structured output generation
using JSON/XML schemas.

Module 2: Production-Grade AI Engineering
Fundamentals

MLOps/LLMOps lifecycle implementation with CI/CD pipelines, data and model versioning,
observability and monitoring systems, AI governance including bias detection and safety filters, and
production deployment strategies.

Module 3: Serverless & Edge AI Deployments
Serverless architecture using FaaS platforms (AWS Lambda, Cloud Functions), streaming responses
and asynchronous processing, security and secret management, edge deployment strategies, cost
optimization, and real-time monitoring dashboards.

Module 4: Community Models & Custom Fine-Tuning

Transfer learning and parameter-efficient fine-tuning techniques (LoRA, QLoRA), navigating open-
source model ecosystem (Hugging Face, Ollama), data preparation for supervised fine-tuning, and

evaluation strategies for custom models.

Module 5: Knowledge Retrieval & Semantic Memory
(RAG)
Embeddings and vector database architecture, RAG pipeline design and optimization, advanced
techniques including hybrid search and re-ranking, quality evaluation metrics, and knowledge graph
integration for structured data.

Module 6: AI Agents & Agentic Workflows
Agent fundamentals with ReAct framework, function calling and tool integration, multi-agent
coordination architectures (CrewAI/LangGraph), reflection and self-correction mechanisms, and
production deployment with monitoring and safety guardrails.

Module 7: Vision-Enabled & Multisensory AI
Diffusion models for image generation, multimodal AI with Vision-Language Models (VLMs), practical
applications including OCR and visual analysis, vision-AI safety considerations, and vision-enabled
agentic workflows.

Module 8: Capstone Sprint
